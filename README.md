# Twitter に掲載されている漫画を読みやすくするやつ

- [リリースしました](https://twmangaview.herokuapp.com)

## できてる

- スレッド形式の連載の先頭ツイートから画像を取得
- 引用形式の連載の先頭ツイートから画像を取得
- 書籍ビューワ
- サムネイル表示
- タイトル/URL 検索
- ユーザ認証
- お気に入り機能
- ユーザの取り込んだ書籍の一覧

## たぶんできてる

- 非画像つきスレッド対策
- 非画像つき引用対策

## 考える

- 書籍の削除

## モチベがあれば

- スレッドの途中からでも取得
- 引用の途中からでも取得
- その他の連載形式に対応

## わからない

- API 上限対策
